--- layout: post title: svnでマージ済みのchangesetを抽出する方法 date: 2014-11-13 tags: [svn] ---

今更ながらのsubversionネタ．あるbranchにコミットされているchangesetが別のbranchにマージ済みかどうかを調べたいことがあった．

すでにマージ済みのchangesetを抽出する方法

{% highlight bash %} $ cd <マージ先のbranch> $ svn mergeinfo --show-revs merged <マージ元branchのURL> {% endhighlight %}

マージしていないchangesetを抽出する方法

{% highlight bash %} $ cd <マージ先のbranch> $ svn mergeinfo --show-revs eligible <マージ元branchのURL> {% endhighlight %}
