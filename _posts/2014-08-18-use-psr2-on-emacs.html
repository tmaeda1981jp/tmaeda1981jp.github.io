--- layout: post title: php-modeでPSR-2準拠でコーディングできるようにする date: 2014-08-18 tags: [php, emacs, flycheck] ---

PHP_CodeSniffer のインストール

{% highlight bash %} $ sudo pear install PHP_CodeSniffer $ phpcs -i The installed coding standards are MySource, PEAR, PHPCS, PSR1, PSR2, Squiz and Zend {% endhighlight %}

flycheckの設定

{% highlight emacs-lisp %} (add-hook 'php-mode-hook (lambda () (setq flycheck-phpcs-standard "PSR2") : {% endhighlight %}

indentの設定

(require 'php-mode) の前に書けばいい？

  • https://github.com/ejmr/php-mode/issues/155
{% highlight emacs-lisp %} (setq php-mode-coding-style (quote psr2)) (require 'php-mode) {% endhighlight %}

とりあえず上記の設定でflycheckの結果 = phpcs xxx.php --standard=psr2 の結果と合致するようになったし，インデントも綺麗にできるようになった．
