--- layout: post title: Emacsでファイル名から拡張子を除いたものを返す関数 date: 2014-08-01 tags: [emacs, elisp] ---

1. file-name-sans-extension を 使う

{% highlight emacs-lisp %} (file-name-sans-extension "/path/to/index.html") ; => /path/to/index {% endhighlight %}

2. f.el の f-no-ext を使う

f.el はファイル関連の処理をいい感じで扱いやすくしてくれるライブラリ．

{% highlight elisp %} (require 'f) (f-no-ext "path/to/index.html") ; => /path/to/index {% endhighlight %}
