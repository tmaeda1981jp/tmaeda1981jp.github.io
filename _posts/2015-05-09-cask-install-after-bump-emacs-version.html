--- layout: post title: emacsをupdateしたあとのcask install date: 2015-05-09 tags: [emacs, cask] ---

自分の環境だと cask install しても古いバージョンのemacs用のパッケージしか取得できなかった。
以下の方法で新しいemacsを明示的に指定してから実行すると新しいパッケージが取得できた。

{% highlight Bash %} $ EMACS="/Applications/Emacs.app/Contents/MacOS/Emacs" cask install {% endhighlight %}
